# GOAT

Listado de todos los goles en la carrera de Lionel Andrés Messi Cuccittini. Diseño por [https://taigatakahashi.com/](https://taigatakahashi.com/)

## Instalación

```bash
git clone https://github.com/brandonporcel/keep-code.git
cd keep-code
npm install
npm run dev
```
